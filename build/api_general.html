Получение токена для api.digiseller.ru

1. Заголовок поменьше

URL и формат запроса:
https://api.digiseller.ru/api/apilogin
Метод:
POST
Заголовки запроса:
Content-Type: application/json
Accept: application/json
Формат запроса JSON:
JSON
XML
{ "retval":"0", "retdesc":"", "id_seller":"1", "id_request":"993812", "img_url":"//shop.digiseller.ru/asp/turing_num.asp?nocash=3021&id=993812" }
Параметры запроса:
Название Назначение Примечание
seller_id идентификатор продавца целое число
timestamp время запроса время запроса
sign Подпись строка, формируется как SHA256 от строки, полученной конкатенацией следующих параметров:
- API_GUID продавца (отправляется в WM-сообщении)
- timestamp (текущее время)
sign=SHA256(API_GUID + timestamp)

GET

Accept: text/xml
Формат запроса JSON:
пример JSON-ответа
{ "retval": 0, "retdesc": "", "inv": 0, "id_goods": 0, "amount": 0.0, "type_curr": "", "profit": "", "amount_usd": 0.0, "date_pay": "", "email": "", "agent_id": 0, "agent_percent": 0, "unit_goods": 0, "cnt_goods": 0, "promo_code": "", "bonus_code": "", "cart_uid": "", "unique_code_state": { "state": 0, "date_check": "", "date_delivery": "", "date_confirmed": "", "date_refuted": "" }, "options":[ {"id": "0", "name": "", "value": ""}, {"id": "0", "name": "", "value": ""} ... ] }
Параметры запроса:
Название Назначение Примечание
retval код выполнения запроса 0 - запрос выполнен
timestamp base64-закодированная строка, содержащая набор параметров, дополнительно переданных с вашего сайта на страницу оплаты oplata.info.
