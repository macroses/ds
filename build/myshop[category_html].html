При желании можно разместить на отдельной странице товары одной категории. Для этого достаточно скопировать код виджета выбранной категории и вставить его на нужную страницу вашего сайта. На одной странице может быть размещен только один код виджета.
HTML-код категории: скопировать в буфер
Если вы хотите, чтобы страница вашего магазина корректно отображалась на мобильных устройствах, добавьте в ее заголовок (<head>) следующий meta-тег:
<metaname="viewport"content="width=device-width, initial-scale=1"/>
